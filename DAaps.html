¿How to use remix?

May 18th, 2022

A decentralized application is software that runs on a peer-to-peer (P2P) blockchain network rather than on a single computer.

One of the must known type of applications on DAap on ethereum is Defi wich consists in financial trasnsactions.The Decentralized applications are the combination of an user interface plus a smart contract Technology. These web applications are designed with html, css and javascript. To be able to interact through the front end you need a wallet like metamask. Metamask will sign the transaction to the blockchain, once the transaction is done the wallet will provide a transaction hash to identify it.

The purpose of the backend is to serve the frontend.

Note

It is important to know during the creation process what will be in the blockchain and what will be outside of the blockchain. Adding a lot of data to the blockchain can be quite expensive.

While you are creating your smart contract you will probably use tools like solidity and remix. You must remember that once you deploy your smart contract into the blockchain you won't be able to change it. That why you should test it. The next step is to create a nice frontend integrating the contract and the wallet. Finally is time to check that everything works fine on the public testnet.

How to use Remix

First you need to go to the Remix , then you need to click in the file sign that will take you to your workspace, once you are there you click in the paper sign to start writing your smart contract, finally you can go to the ethereum sign to deploy your smart contract.

How to deploy your smart contracts Remix

Once you have created your smart contract you will probably want deploy it. You can start by compiling it pressing Ctrl + s, to verify that everything is in order. Then you go to the deploy space that we checked before. First there is a space where you can select where would you like to deploy it, if you are only testing it is okay to keep the enviroment in the javascript VM, the account is to pick the address you are going to be using. The gas limit and the value you can leave them like that if you are not quite sure of how this works. Finally you can press the deploy botton. Finally you should see an information of the transaction you just made.
