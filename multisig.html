How to create a multisig wallet?

May 23th, 2022

This is a type of wallet that can only send ether if multiple ethereum adresses agree or review the transaction with a signature. This means that is needed more than one key to authorize the transaction in the blockchian. In terms of security having multiple keys upgrades the protection of the wallet, so if you hold significant amount of cryptocurrency you should consider the idea of using multiple keys. In this section we are going to see how can we make this possible with solidity.
